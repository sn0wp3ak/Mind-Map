4. Format Output

placeholder:

print("----%s----%d----" % (var1, var2))


format function:

place ordered

1.print("----{}----{}----".format(var1, var2))

number ordered

2.print("----{0}----{1}----".format(var1, var2))

keyword

3.print("----{a}----{b}----".format(a=var1, b=var2))


for %s the f string is better!

a = "apple"

b = "banana"

print(f"{a}, {b}")
